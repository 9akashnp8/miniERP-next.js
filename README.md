# About

React Frontend for [miniERP](https://github.com/9akashnp8/mini-ERP) built with Next.js

# Getting Started
Running then development server:
```
git clone https://github.com/9akashnp8/miniERP-next.js.git
cd miniERP-next.js
npm run dev
```
